Swipe.js

Swipe.js is a lightweight JavaScript app (JQuery/Angular.js compatible) that allows you to create a slideshow of elements that can be triggered by both clicking and swiping. Below is a example of the usage of slideshow. The original demo link can be found here!

In the demo the I customized below, the biggest change I added is to change the color of the slide show by button color. The default color is orange but it changes corresponding to the color of the button clicked based on JavaScript manipulation. I also added enable and disable button to enable/disable swipe on slideshow as well as button click to navigate slideshow (you can still change colors)! Refer to this readme document for more information and usage!
