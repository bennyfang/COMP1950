CSS3 Techniques

Techniques demonstrated

  1. Border-radius & Box-shadow - Refer to image in My Favourite Game section. It has a border radius of 10% and border-shadow on bottom and right!
  2. Text-shadow - this is text-shadowed!
  3. Transition - Header and footer utilizes transition properties. Vendor prefixes are used here for compatibility to all browsers.

New techniques not demonstrated in class

  1. Stroke - A CSS3 property that supports the new svg tags introduced in html5 that allows the browser to draw shapes such as the circle below. The stroke properties are used (stroke, fill, stroke-width) to style the circle!

  2. Text-overflow - A CSS3 technique that allows overflowing text to be hidden in a box-model container and then represented by feedback such as ellipsis. In the example below referenced here, I use text-overflow property to first hide text and then expand them upon hovering. I also added a 0.5s transition effect as well. Interesting fact- this is actually developed by Internet Exploder and is compatible with IE since IE6!!
