Useful Tools

  • Bootstrap - a popular html, css and js open source library that support responsive design and useful prebuilt components.
  • JQuery - a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.
  • AngularJS - a Java Script-based open source front end web application framework mainly maintained by Google. It supports dynamic views in web-applications and lets you extend HTML vocabulary for your application.
  • ReactJS - a JavaScript library for building interactive user interfaces. It allows developers to create large web-applications that use data and can change over time without reloading the page and aims primarily to provide speed, simplicity, and scalability.
  • ASP.NET MVC - a web application framework developed by Microsoft, which implements the model–view–controller (MVC) pattern. It is a powerful, patterns-based way to build dynamic websites that enables a clean separation of concerns and that gives you full control over markup for enjoyable, agile development. ASP.NET MVC includes many features that enable fast, TDD-friendly development for creating sophisticated applications that use the latest web standards.
